# 🌐 Websites

A collection of web projects built with vanilla HTML, CSS, JavaScript, and PHP.

---

## 📁 Projects

### 📚 Library Management System

A full-stack web application for managing a library — books, members, and borrowing records.

**Features:**
- Add, edit, and delete books
- Manage library members
- Track borrowed and returned books
- Search and filter book catalog
- Admin dashboard

**Tech Stack:**

| Technology | Purpose |
|---|---|
| PHP | Backend logic & server-side rendering |
| MySQL | Database |
| CSS | Styling & layout |
| JavaScript | Frontend interactivity |

## 🚀 Getting Started

### Prerequisites
- [XAMPP](https://www.apachefriends.org/) or any local PHP server (Apache + MySQL)

### Run Locally

1. Clone the repo:
```bash
git clone https://github.com/ManiINFINITE/Websites.git
```

2. Copy the project folder into your XAMPP `htdocs` directory:
```
C:/xampp/htdocs/Library Management System/
```

3. Import the database:
   - Open **phpMyAdmin** at `http://localhost/phpmyadmin`
   - Create a new database
   - Import the `.sql` file from the project folder

4. Open your browser and go to:
```
http://localhost/Library Management System/
```
